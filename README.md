# intelligenism.club

Static site for Intelligenism Agent Framework, v4 Protocol, and Community.

## Structure

```
index.html          — Main page (auto-discovers articles from pages.json)
pages.json          — Article registry (auto-generated by build_pages.py)
build_pages.py      — Scans articles/, generates pages.json
articles/           — Individual HTML pages (docs + updates)
.github/workflows/  — GitHub Action: auto-rebuild pages.json on push
```

## Add a new article

1. Create `articles/your-article-name.html`
2. Include meta tags in `<head>`:
   ```html
   <title>Your Article Title</title>
   <meta name="article-date" content="2026-03-30">
   <meta name="article-category" content="update">
   <meta name="article-description" content="Short description for index card.">
   ```
3. `git push` — GitHub Action auto-updates `pages.json`

## Categories

- `doc` — Permanent documentation (IAF architecture, v4 spec, etc.)
- `update` — News/changelog (shown in Updates feed on index)
- `release` — Version releases (also shown in Updates feed)
